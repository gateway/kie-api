# Codex Skills

These are repo-bundled Codex skill definitions in the folder-based `SKILL.md` format used by Codex skill bundles.

Included skills:
- `find-latest-media`
- `check-credits`
- `nano-banana`
- `kling-video`
- `chain-image-to-video`

To install them into your local Codex skills folder:

```bash
python3 scripts/install_codex_skills.py
```
