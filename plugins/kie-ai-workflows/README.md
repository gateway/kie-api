# Kie.ai Workflows Plugin

This Codex plugin packages the Kie.ai workflow skills from this repo into an installable plugin structure.

## What it does

It gives a Codex user a simple first-run path for:
- setting up `KIE_API_KEY`
- checking credits
- running a first Nano Banana image test
- running Kling video flows
- chaining image-to-video runs
- retrieving the latest generated media from the artifact library

## First-run path

1. Set up a Kie.ai key.
2. Check credits.
3. Run a small Nano Banana test.
4. Move on to Kling or chained workflows.
